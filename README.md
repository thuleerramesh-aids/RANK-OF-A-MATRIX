# RANK-OF-A-MATRIX
## Aim:
To write a python program to find the rank of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step 1: Start
Import the NumPy library.
### Step 2: Input Matrix
Define the matrix (e.g., matrixA = [[1,2,3],[3,6,9]]).
### Step 3: Compute Rank
Use np.linalg.matrix_rank(matrixA) to calculate the rank.
### Step 4: Display Result & Stop
Print the rank value and end the program.
## Program:
```
#Program to find the rank of a matrix.
#Developed by: THULEER.R
#RegisterNumber:212225230285
import numpy as np
matrixA =np.array([[1,2,3],[3,6,9]])
rank = np.linalg.matrix_rank(matrixA)
print(rank)
```
